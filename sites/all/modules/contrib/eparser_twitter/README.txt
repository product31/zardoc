eparser_twitter is an extension to the xml parser of feedapi_eparser.

Instructions:
1. Install the eparser_twitter module
2. Create a feed node with the URL pointing to your twitter
   xml URL and the feed item node type as your tweet node type.
3. Click the Eparser tab of your feed node and select twitter
   as your feed type.
4. If you have feedapi_mapper, map the twitter title field to your
   title field.
