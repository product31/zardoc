Translatablecomments is a module that adds buttons to comments on your site for verious different languages. Clicking on one of the languages will translate the comment using the Google AJAX Language API.

Installation
------------

Copy the translatablecomments folder to your module directory, then enable it at Administer > Site Building > Modules.
