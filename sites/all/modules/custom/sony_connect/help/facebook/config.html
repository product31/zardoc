1. Get started by logging into Facebook with the D2C Connect (d2c.connect@sonymusic.com) account.
  2. After logging in, click Applications in the bottom toolbar, then click on the Developer application.
  3. Once inside the developer application click the Set Up New Application button near the top right of the page.
  4. Name the application the same as the site, agree to the Terms and click the Save Changes button.
  5. Back on this site, copy and paste the API Key and Secret into the textfields on the configuration form. Click Save configuration on the form.
  6. Now the site will auto-configure the Facebook Application with all of the necessary settings!
