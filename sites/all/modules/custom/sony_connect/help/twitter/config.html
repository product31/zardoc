1. Get started by logging into the Twitter Applications section with the D2C Connect (d2c.connect@sonymusic.com) account.
  2. After logging in, click the Register a new application link at the bottom of the applications list.
  3. Upload the 75x75 application icon, and use the following values to fill out the form:
    Application Name:
    The Official [Artist Name] Site
    Application Website:
    The full domain of the live site. ex: http://www.example.com/
    Organization
    Sony Music
    Website
    http://www.sonymusic.com/
    Application Type
    Browser
    Callback URL
    http://www.example.com/connect/twitter/post_authorize
    Default Access Type
    Read & Write
    Use Twitter for login
    Yes
  4. Back on this site, copy and paste the Consumer Key and Secret into the textfields on the configuration form. Click Save configuration on the form.
